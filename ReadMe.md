**Copy Environment file:**
- Copy `.env.example` to `.env` in server folder
- Update the following fields:
```
JWT_SECRET=your_secret_key
PORT=5000
```

**Frontend Setup**
```bash
cd client
npm install
npm run dev
```
Frontend will run on: **http://localhost:5173**

**Backend Setup**
```bash
cd server
npm install
node index.js
```
Backend will run on: **http://localhost:5000**
